Detailed Description

Abstract base for full-text converters.

A Converter accepts a complete UTF-8 string and returns the converted result. Internally it coordinates Segmentation (splitting text into words) and a ConversionChain (applying one or more dictionaries to each segment). Concrete implementations:

  • SingleStageConverter — one segmentation pass + one ConversionChain.
  • PipelineConverter — passes text through a sequence of Converter stages in order, each with its own segmentation and chain.

Config-backed converters may additionally expose the main conversion chain via GetConversionChain() even when an internal normalization pre-pass is present.
